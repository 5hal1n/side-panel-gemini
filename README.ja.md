# side-panel-gemini

## プロジェクト概要
これはChrome拡張機能プロジェクトです。

## インストールガイド
このChrome拡張機能をインストールするには、以下の手順に従ってください。

1.  **リポジトリをクローンする（またはソースコードをダウンロードする）：**
    Gitがインストールされている場合は、リポジトリをクローンできます。
    ```bash
    git clone https://github.com/shalin/side-panel-gemini.git
    cd side-panel-gemini
    ```
    または、`src`ディレクトリの内容を手動でダウンロードしてください。

2.  **Chromeの拡張機能ページを開く：**
    Google Chromeブラウザを開き、`chrome://extensions`にアクセスします。

3.  **デベロッパーモードを有効にする：**
    拡張機能ページの右上にある「デベロッパーモード」のスイッチをオンにします。

4.  **パッケージされていない拡張機能を読み込む：**
    表示される「パッケージされていない拡張機能を読み込む」ボタンをクリックします。ファイルダイアログが開きます。

5.  **`src`ディレクトリを選択する：**
    プロジェクトをクローン/ダウンロードした場所から`src`ディレクトリに移動し、選択します。

読み込みが完了すると、`side-panel-gemini`拡張機能がインストールされている拡張機能のリストに表示され、使用できるようになります。
